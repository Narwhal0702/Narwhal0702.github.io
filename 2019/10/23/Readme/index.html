稀疏数组与二维数组的转换
Created 2019-10-23| Updated 2019-11-30|DataStructure
Post View:

稀疏数组转换成二维数组

  1. 遍历原始的二维数组，得到有效数据的个数sum
  2. 根据sum创建稀疏数组sparseArr int[sum+1][3]
  3. 将二维数组的有效数据存入到稀疏数组中

二维数组还原成稀疏数组

  1. 先读取稀疏数组的第一行，根据原始的二维数组，创建原始的二维数组
  2. 读取稀疏数组后几行的数据，并赋值给原始的二维数组
